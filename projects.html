Huffman Coding

This project is a classic. To save space when sending documents, you can use Huffman Coding to shrink the size of the doc. This project uses a variety of data structures, including maps, trees, and priority queues in C++. Taking a given string, this project will transform the text into binary and then converts it back to text.
